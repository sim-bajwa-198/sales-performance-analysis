# sales-performance-analysis
Sales performance analysis using Excel
Project Overview
This project analyses sales data using Excel to identify revenue trends, top-performing products, categories, and regions.

Business Questions
- What is total revenue?
- How does revenue change over time?
- Which prodcuts and categories perform best?
- Which regions generate the most revenue?

Dataset
Publicly available sales transaction data containing order dates, products, categories, regions, quantities, and revenue.

Tools Used
- Microsoft
- Pivot tables
- Charts

Key Insights
- Revenue peaked in Q4 2024, indicating seasonal demand.
- Furniture and Home Appliances drove the majority of revenue.
- Asia was the highest-performing region.

Recommendations
- Increase inventory and marketing ahead of peak seasons.
- Focus growth efforts on high-performing regions.
- Review low-performing products and regions for optimisation.

Notes
- Quantities were rounded to whole units for realism.
- Currency was not specified and treated as a generic revenue amount.
